# AfterTheFall
Re:Mind — After the Fall (RMATF) bringt einen „After-the-Fall“-Look für Project Zomboid (Build 42) durch deterministische World-Mutation beim Chunk-Load: Overgrowth/Unkraut, Wandranken, gelegentliche Bäume (optional ohne Straßen), plus dezenter visueller „Decay“ (z. B. Dach-/Wandschäden). Steuerbar über umfangreiche Sandbox-Optionen.
